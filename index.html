Repitition VR

Using Unreal Engine 5, my team and I created a virtual reality game that aims to combat social isolation and anxiety. The games centers dialogue interaction in the scene and solving puzzles in order to progress from level to level. It includes 4 different scenes and was completed over the course of 4 weeks. Features included are: VR dialogue mechanics, 3 puzzles with separate variations, and 3D animation. Using Blender, I modeled and animated the "narrator". Visit our website here!

A2Go!

Using Unity, my partner (Sunidhi Majalikar) and I created a Pokemon Go! inspired game with augmented reality features in order to reinstate Ann Arbor as the best tree city in the U.S. Using the game's AR mode, the player must plant trees around the city, ward off squirrels, and collect petals to buy seedlings. Visit landmarks to unlock seeds and visit the cemetery for a special challenge! This project includes various features such as: object spawning, in-game "combat", landmarks, image tracking, and location precision.
